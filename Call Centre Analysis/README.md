# Call Centre Analysis

## Overview
This project analyzes call centre performance data to uncover trends in customer service efficiency and satisfaction.

## Tools & Skills
Power BI (dashboard development and visualization)
Power Query (data cleaning and preprocessing)

## Dataset
PwC Call Centre dataset

## Process
1. Cleaned and validated raw call records to ensure accuracy
2. Built Power BI dashboard to visualize call volumes, wait times and abandonment rates

## Key Insights
1. 19% of calls were abandoned before being answered
2. Average speed of answer was 68 seconds
3. Longer call durations correlated positively with higher customer satisfaction

## Recommendation
Reduce wait times and improve staffing allocation during peak hours.
